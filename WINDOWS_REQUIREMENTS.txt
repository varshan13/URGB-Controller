FIRST RUN SETUP
===============

Step 1: Install Hardware Software
---------------------------------
Install the appropriate software for your hardware:
‚Ä¢ MSI Center (MSI motherboards)
‚Ä¢ Razer Synapse (Razer peripherals)
‚Ä¢ OpenRGB Server (universal support)
‚Ä¢ Vendor-specific software as needed

Step 2: Start Background Services
---------------------------------
Ensure these services are running:
‚Ä¢ OpenRGB Server (openrgb.exe --server)
‚Ä¢ MSI SDK service (if using MSI hardware)
‚Ä¢ Razer services (if using Razer hardware)

Step 3: Launch RGB Control Center
---------------------------------
1. Run as Administrator (required for hardware access)
2. Click "Scan Devices" to detect hardware
3. Configure device settings in the interface
4. Set up 3D device placement if desired

Step 4: Create Your First Profile
---------------------------------
1. Set colors and effects for each device
2. Adjust brightness and speed settings
3. Save profile with a descriptive name
4. Test synchronization across all devices
